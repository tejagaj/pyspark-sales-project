# PySpark Sales ETL Pipeline

## Overview
This project demonstrates a production-grade PySpark ETL pipeline with:
- Modular design
- Pylint-compliant code
- Pytest-based unit testing

## Project Structure
- `spark_session.py`: Spark session creation
- `sales_transformations.py`: Business logic
- `sales_job.py`: Pipeline orchestration
- `tests/`: Unit tests for transformations

## Run Tests
```bash
pytest
